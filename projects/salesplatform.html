Air Cargo Sales platform

Tech Stack - HTML+SASS+Javascript+ReactJS, Java+SpringBoot, AWS RDS for PostgreSQL, AWS Elastic Beanstalk, AWS Simple Queue Service(SQS), AWS Simple Notification Service(SNS)

Problem - No elegant fully-digital, one-stop-shop of booking air cargos as well as tracking them.
Solution - A sales platform for cargo business which aggregates flights from multiple carriers and presents an intuitive, responsive UI for searching, filtering, comparing and booking flights, as well as tracking cargos on the go.
A cross-platform flight booking engine that makes air cargo booking as easy as booking holidays on hotel aggregators, with the flexibility of onboarding other modes of cargo to the platform in the future.

Standout Features

  • Extensive Searchability with Filtering: Search not only based on source, destination, dates but also on product type, weight, dimensions,etc.
  • Recent Searches: Auto save and load of recent searches for a user to make the experience more seamless.
  • Flight Compare: Users can select upto 4 flights to view them more granularly to reach a decision.
  • Save or Book: Users have the ability to do end-to-end booking but also has the priviledge to save and resume booking at a later point of time.
  • Cargo Tracking: Real time tracking for air cargos with proper messaging in case of delays or cancellations.
  • Power Gantt visualization: Flights stacked as Gantt charts over a timeline which provides a great view to the user on duration and number of stops.
  • Cross Platform support: Looks great on desktop, tablet as well as mobile.
