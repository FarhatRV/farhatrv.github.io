Meetrio - Meeting Notes and Audio Sync

Link to extension in Chrome Web Store
Tech Stack - HTML+SASS+Javascript+ReactJS, AWS DynamoDB, AWS Lambda, AWS Simple Storage service(S3), AWS API Gateway, AWS CodeBuild(CI/CD), Socket.io

Problem - Textual notes are not detailed enough, audio recordings take too long to consume.
Solution - Developed a Chrome extension to sync meeting notes(taken in Google Docs) and audio.
This is a chrome extension to capture user media(here, audio) and sync it with notes taken in Google Docs and provide a mechanism to jump to an audio when a user selects a text in the Google Doc.

Standout Features

  • Distributed note taking: Take notes simultaneously with the team members using Google Docs.
  • Secure cloud storage: Record your meeting from your browser. Audio recordings are saved securely in the Cloud.
  • Precision recall: Link to the exact point in the audio recording by selecting the text of interest.
  • Multi-session notes: Continue note-taking in the same document over several sessions.
  • Offline sync: Sync textual notes to audio file at a later point.
  • Offline edits: Edit notes after the meeting without affecting the text to audio sync done earlier
