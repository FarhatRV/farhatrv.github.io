Team Secrets - Secure file sharing for Jira and Gmail

Official Website - teamsecrets.io
Tech Stack - Java+Spring+Hibernate, HTML+SASS+Javascript+AngularJS, AWS RDS, AWS KMS, AWS Simple Storage service(S3)

Developed a plugin that makes it easy to protect sensitive files with end-to-end encryption and viewer verification. Team Secrets uses bank-grade encryption to protect your file attachments from the moment you start uploading until they are opened by the recipient. Snooping on the transfer or stealing the file will result in something completely undecipherable.

Team Secrets for Jira

  • Encrypt files and text fields right inside your Jira issues.
  • Restrict access to only a limited set of people or group of people.
  • Lets you choose your own AWS S3 bucket for storing encrypted secrets.
  • Jira admins can add a custom encrypted field to any issue type.
  • Audit log - Track every action and attempted access for your secrets.

Team Secrets for Gmail

  • Encrypt file attachments in Gmail right from your inbox.
  • Require 2-factor authentication using a recipient's mobile number.
  • Add an extra layer of security by requiring a passphrase.
  • Set an expiration timer in days, weeks or months.
